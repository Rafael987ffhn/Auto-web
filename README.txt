
# AutoDub Web (com ffmpeg-python)

## Como executar localmente:

1. Instale as dependências:
```
pip install -r requirements.txt
```

2. Execute o servidor:
```
python app.py
```

3. Acesse no navegador:
```
http://localhost:5000
```

Certifique-se de ter o FFmpeg instalado no seu sistema e acessível pelo terminal.
